# FileCryptoGolang

## Usage:
  -b string
    	File to bruteforce
  -c string
    	Characters for bruteforcing
  -d string
    	File to decrypt
  -e string
    	File to encrypt
  -l int
    	Max password length for brute force
  -o string
    	File to save decrypted content into
  -p string
    	Password to encrypt/decrypt file
